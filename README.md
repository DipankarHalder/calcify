## Calcify

A medical portal, where doctor and patient can communicate with each other easily

Modules:
1. Doctor OnBoard
2. Patient Registration and Login
3. Appointment
4. Diet tracker
5. Medicine tracker
etc...

Tech Stack:
1. React.js
2. Next.js
3. Redux-toolkit
4. Redux-thunk
5. Node.js
6. Express.js
7. Joi
8. Pritter
9. ESlint
10. JavaScript
11. ES6 / ES7
12. Async await
